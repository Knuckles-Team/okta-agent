# okta-agent

Enterprise CIAM/SSO connector: MCP server + Pydantic AI agent over the Okta
Management API.

- **[Overview](overview.md)** — architecture, tool taxonomy, safety model
- **[Installation](installation.md)** — pip extras, environment variables
- **[Usage](usage.md)** — API client, CLI, MCP tool examples
- **[Concepts](concepts.md)** — concept registry (`CONCEPT:OKTA-*`)

`okta-agent` complements `keycloak-agent`: the same verb taxonomy over the
commercial IdP, so agents can switch identity providers without relearning
tools.
